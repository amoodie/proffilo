.. api.profile:

*********************************
Vertical profile classes
*********************************

The package makes available models for vertical profiles as classes.

All classes inherit from :obj:`BaseProfile`, and redefine methods and attributes for their own purposes.

The classes are defined in ``proffilo.profile``.
