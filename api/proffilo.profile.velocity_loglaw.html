velocity_loglaw¶

proffilo.profile.velocity_loglaw(z, z0, ustar, alpha=1)¶

Compute velocity profile following the log-law (Law of the Wall) velocity profile.

:cite:`Strunk1979`

\[z0 = ks d_{90}\]
Parameters

d90 (float) – The 90th percentile of the bed material grain-size distribution, given in meters.

Returns

The roughness height \(z_0\).

float
