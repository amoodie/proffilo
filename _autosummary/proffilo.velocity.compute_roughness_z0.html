proffilo.velocity.compute_roughness_z0¶

proffilo.velocity.compute_roughness_z0(d90)¶

Compute roughness for velocity profile.

Computes the rougness height for the velocity profile based on the formula:

.. math:: z_0 = k_s d_{90}
Parameters

d90 (float) – The 90th percentile of the bed material grain-size distribution, given in meters.

Returns

The roughness height .

Return type

float
