show_profile(block=False, save_str=None, return_ax=False, **kwargs)¶

Show the profile.

Show the profile in a plot.

Parameters

  • block (bool, optional) – Whether to pause script execution by showing the plot. I.e., the block argument in matplotlib’s plt.show().

  • save_str (str, optional) – String to save the output file.

  • return_ax (bool, optional) – Whether to return the axis object; default is False. If True, block and save_str are ignored and the axis is returned before saving or showing.

  • **kwargs (optional) – Any arbitrary matplotlib.pyplot.plot() keyword arguments for the plot specification. Note that these specs are passed to all lines.

Returns

ax – The axis object. Only provided if parameter return_ax=True.

Return type

matplotlib.pyplot.axes
