proffilo.observation.VelocityObservations¶

class proffilo.observation.VelocityObservations(data=None, flow_depth=None, connection={})¶

Velocity concentration observations collection.

Note

There is no support for indexing with original column labels via pd.DataFrame methods. For example, if an Observation class is initialized with the 'concentration' field as 'conc' in the dataframe, but linked via connection, you cannot do stn.conc_obs.loc[:,'conc'].

__init__(data=None, flow_depth=None, connection={})¶

Initialize a VelocityObservations object.

Parameters

  • data (pandas.DataFrame, optional) – An optional DataFrame with the observation data preconfigured. If not supplied, this field is set to None, and data can be filled in via the add_observation method.

  • depth (flow) – Flow depth [m]. Some functionality may be limited if this is not supplied.

show_profile(block=False, save_str=None, return_ax=False, **kwargs)¶

Show observations as a profile.

Parameters

  • block (bool, optional) – Whether to pause script execution by showing the plot. I.e., the block argument in matplotlib’s plt.show().

  • save_str (str, optional) – String to save the output file.

  • return_ax (bool, optional) – Whether to return the axis objectl; default is False. If True, block and save_str are ignored and the axis is returned before saving or showing.

  • **kwargs (optional) – Any arbitrary matplotlib.pyplot.plot() keyword arguments for the plot specification. Note that these specs are passed to all lines.

Returns

ax – The axis object. Only provided if parameter return_ax=True.

Return type
