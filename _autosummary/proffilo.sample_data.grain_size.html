proffilo.sample_data.grain_size¶

Sample grain-size distributions.

There are seven grain size distributions included in this package as samples. Each distribution is characterized by an Mx2 ndarray, where the first column includes the grain size bins, and the second is a cumulative distribution.

Example

Grain size samples can be instantiated as, for example:

>>> import proffilo as pf
>>> _yrdata = pf.sample_data.grain_size.yellow_2015()

Note

Be sure to use is_cumulative=True if instantiating Distribution objects with these datasets.

Available information on the grain size distributions is enumerated in the following section.

Grain-size distributions¶

brazos()ndarray

This distribution is from the Brazos River, Texas, USA.

amazon()ndarray

This distribution is from the Amazon River, Brazil.

mississippi_SL()ndarray

This distribution is from the Mississippi River, USA. It was collected near St. Louis.

mississippi_NO()ndarray

This distribution is from the Mississippi River, USA. It was collected near New Orleans.

yangtze()ndarray

This distribution is from the Yangtze River, China.

yellow_1980()ndarray

This distribution is from the Lower Yellow River, China. It was collected in 1980.

yellow_2015()ndarray

This distribution is from the Lower Yellow River, China. It was collected in 2015.
